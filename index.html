JS & JQuery Flashcards

JavaScript (JS)

A programming language that enables developers to give websites interactive features

Operators

Symbols that perform basic operations in JS, such as +, <, ===, *

Variables

Objects in JS that hold a place in a function, allowing data to be passed into that function by a programmer or user

Variable naming conventions

Guidelines for naming variables to make code more readable. JS variables are named in camelCase and should be given names that clearly indicate their purpose without being too long.

Functions

A set of instructions defined in code and given a specific name and parameters, so that those instructions can be called efficiently later and used multiple times with different arguments.

Method

Built-in "mini-functions" that can be used to edit and parse strings.

Arguments

The data passed to a function that meets the requirements of the function's parameters.

Parameters

The variables on which a function is executed.

Return

The output generated by a function after it is performed.

Chaining method

Also nesting, calling one function after another on an object in one line of code.

Strings

A JavaScript string stores a series of characters

Boolean

A data type that encodes binary True or False values

Undefined

JavaScript assigns 'undefined' to any object that has been declared but not initialized or defined.

NAN

The NaN property represents "Not-a-Number" value, such as the result of dividing by zero. However, JS treats it as a number and not a string.

Escape

The escape() function encodes a string by replacing special characters that JS cannot handle with hexadecimal-encoded substitutes.

Alert

The alert() method displays an alert box with a specified message and an OK button.

Comments

Normally starting with // or "< !-- -->"
Is used to write reminders or to block a section of code off.

jQuery

jQuery is a JavaScript library designed to simplify HTML DOM tree traversal and manipulation,
as well as event handling

Attributes

All HTML elements can have attributes.
Attributes provide additional information about an element
